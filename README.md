# Psychology Paper Alert

심리학 논문을 실시간으로 추적하고 알림을 받는 웹 애플리케이션입니다.

## 기능

- ✅ 키워드 기반 논문 검색
- ✅ Crossref API를 통한 실시간 논문 검색
- ✅ 심리학 관련 논문 자동 수집
- ✅ 논문 목록 표시 및 관리
- ✅ 알림 설정 및 관리

## 사용 방법

1. 키워드 입력 (예: depression, anxiety, therapy)
2. 알림 추가 버튼 클릭
3. 자동으로 관련 논문 검색 및 표시
4. 논문 링크를 통해 상세 정보 확인

## 기술 스택

- HTML/CSS/JavaScript
- Crossref API
- Local Storage
